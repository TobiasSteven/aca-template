Bloc WordPress Témoignages

Démonstration du bloc personnalisé de témoignages pour le thème ACA

Fonctionnalités du bloc

  • Titre de section personnalisable
  • Témoignages multiples avec possibilité d'ajouter/supprimer
  • Upload d'images ou URL directe pour les photos des témoins
  • Citations éditables avec formatage riche
  • Nom et fonction/lieu personnalisables
  • Design responsive et moderne
  • Couleurs adaptées au thème ACA
  • Effets de survol élégants

Comment utiliser ce bloc dans WordPress

  1. Dans l'éditeur Gutenberg, cliquez sur le bouton "+" pour ajouter un bloc
  2. Recherchez "Témoignages" ou naviguez vers la catégorie "Mon Thème ACA"
  3. Cliquez sur le bloc "Témoignages" pour l'ajouter
  4. Utilisez le panneau latéral pour configurer chaque témoignage :
    • Uploader une image ou saisir une URL
    • Éditer la citation directement ou via le panneau
    • Modifier le nom et la fonction
    • Ajouter/supprimer des témoignages selon vos besoins
  5. Personnaliser le titre de la section si nécessaire
